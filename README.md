1. task_1
    строка содержит скобки открывающиеся и закрывающиеся, она является нормализованной если 
   скобки открываются и закрываются в правильном порядке
   
   Пример: 
   g{fsdfg}s - true
   g{f(sd)fg}s - true
   {g{fsdfg}s - false
   {{[cc(c)ggg]}ssss} - true
   {{[cc(c)ggg]}ssss}{ - false
